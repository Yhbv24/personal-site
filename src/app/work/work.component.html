Bob Ross Data Visualization

Project Goal

The goal of this project was to expand my knowledge of the D3 library for JavaScript in a fun way. It was built by a team of six, and required us to parse data from a CSV into a JSON, and then write a custom function to turn each episode's data into aggregate data from each season.

Once we had the data, we were able to turn it into a bar graph, line graph, and scatterplot using D3.

Technologies Used

  • JavaScript / Typescript | Primary Language
  • Angular 2 | MVC
  • Firebase | Database
  • D3 | Data Visualization Library
Hockey Team Manager

Project Goal

The goal of this project was to learn how to create a web app with full CRUD functionality using a Firebase database. It was meant to be a team management application, where a "manager" can use a password-protected section to add/edit/remove staff and players, and everyone else can view the roster.

Each person on the roster has his/her own tile, which can be clicked on to see more about that particular person using dynamic routing. Since I have an interest in hockey, I thought this would be a fun way to learn how to create the app.

Technologies Used

  • JavaScript / TypeScript | Primary Language
  • Angular 2 | MVC
  • Firebase | Database
Shoe Store Manager

Project Goal

The goal of this project was to learn how to create a web app with full CRUD functionality using PHP, MAMP, and an MVC to help control the data and keep everything organized. It makes use of a many-to-many relationship using a table for shoe brands, a table for shoe stores, and a join table to connect the two.

Technologies Used

  • PHP | Primary Language
  • Silex | MVC
  • Twig | Templating Engine
  • mySQL | Database
